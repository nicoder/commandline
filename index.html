Command line tips and tricks

pour git bash, la pic, ...

navigation

  • `Ctrl-A` pour aller au début de la ligne
  • `Ctrl-E` pour aller à la fin de la ligne
  • `Ctrl-W` pour effacer le mot précédent le curseur
  • `Ctrl-U` pour effacer toute la ligne
  • `Ctrl-Y` pour coller ce qu'on a effacé

history

  • `Up` / `Down`
  • Ou `Ctrl-p` / `Ctrl-n`

`Ctrl-R`

Pour rechercher une commande

  • `Ctrl-R` de nouveau pour chercher plus loin en arrière
  • `Ctrl-S` pour revenir sur ses pas
  • puis Enter pour exécuter la commande
  • ou les flèches pour la modifier

Par exemple : `Ctrl-R merge` => # git merge hotfix/1234-bug-complexe --no-ff
